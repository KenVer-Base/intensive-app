**Intensive App** adalah sebuah **landing page modern dan responsif** untuk layanan *Intensive Bimbingan Belajar*.  
Project ini dibuat menggunakan **React**, **Vite**, dan **Tailwind CSS**, serta difokuskan pada tampilan yang bersih, ringan, dan mudah dikembangkan.
